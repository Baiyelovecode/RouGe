# 数值之塔 CLI 原型

一个参考《杀戮尖塔》的命令行文字肉鸽爬塔原型。

## 运行

```powershell
cd C:\baiye-uav\tower-rogue-cli
python main.py
```

## 已实现

- 3 章 x 8 层，共 24 层
- 手动回合制战斗
- 能量、护甲、力量、敏捷
- 易伤、虚弱、中毒、灼烧
- 敌人意图
- 战斗后 3 选 1 技能奖励，可跳过
- 精英、Boss、宝箱掉落遗物
- 商店、休息、随机事件

## 战斗命令

- 输入技能编号：使用技能
- `end`：结束回合
- `log`：查看最近战斗日志
- `skills`：查看技能池
- `relics`：查看遗物

## 设计说明

首版优先验证战斗和构筑选择，不做完整卡组抽弃牌机制。每回合会从当前技能池随机展示最多 5 个技能，保留“回合选择”和“技能池污染”的核心策略。
# 图鉴

启动游戏后可从主菜单进入卡牌图鉴或遗物图鉴。卡牌图鉴同时展示 Lv.0、Lv.1、Lv.2 的费用与效果，遗物图鉴会标注通用或角色专属。

也可以直接打印完整图鉴：

```powershell
python main.py --cards
python main.py --relics
```
